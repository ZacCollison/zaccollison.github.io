NBA Tableau Analysis

Analytics and Basketball are great interests of mine. I wanted to develop an analysis on the impact of players in NBA teams using using Tableau, as I'm intrigued by the story data/numbers can present and I am obsessed with basketball. The analysis uses a range of techniques and presenation features to understand the data such as, clustering algorithms, bar/ line graphs and a map display feature for presenation. The Analysis also includes a search bar and key on the side to search for specific player stats within the database.

Stock data interface

I created this project out of curiosity of the yfinance module and was interested in obtaining real-time data. I use an object oriented approach to the project to organise my methods and ticker information for the user interface. Furthermore, i use the matlabplot library to create figures to draw into a simple gui for a user to interact with. I created this gui using the PySimpleGUI module in python, using simple buttons, a canvas and multiple text sections in the layout.
